 ## React TicTacToe Game with Computer
 
Tic-Tac-Toe is played on a square grid of 3Ã—3 squares. Two players, compete against each other. They must each in turn fill in a square on the grid with the symbol assigned to them: O or X.

The winner is the one who manages to align three identical symbols, horizontally, vertically or diagonally.
